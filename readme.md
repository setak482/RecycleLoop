# ♻️ RecycleLoop (재활용 오브젝트 루프 시퀀서)

**RecycleLoop**는 재활용품(유리병, 캔, 종이박스 등)의 소리를 악기로 활용하여 음악을 만들 수 있는 **웹 기반 단일 채널 루프 시퀀서 및 피아노롤(Piano Roll)** 입니다. 버려지는 물건들의 소리에 생명을 불어넣어 자신만의 독창적인 루프 패턴을 손쉽게 작곡할 수 있습니다.

### 🔗 [Live Demo 보러가기](https://recycle-loop-maker.vercel.app/)

<br>

## ✨ 주요 기능 (Features)

*   **재활용품 악기 기반 시퀀싱:** 빈 병 불기, 유리잔 부딪히기, 빈 캔 두드리기 등 고유한 오디오 샘플(`Tone.js` 활용) 지원
*   **직관적인 피아노롤 인터페이스:** 마우스를 이용한 노트 배치/길이 조절 및 자유로운 화면 줌(Zoom) & 팬(Pan) 기능
*   **재생 및 편집 제어:** 루프 구간 반복 설정, 트랜스포트(재생/정지) 컨트롤
*   **고급 편집 툴:** 영역 선택, 복사/붙여넣기, 작업 내역 기록을 통한 실행 취소/다시 실행(Undo/Redo)
*   **프로젝트 관리:** 작업 중인 패턴 데이터 저장 및 불러오기

<br>

## 🛠 기술 스택 (Tech Stack)

*   **Frontend:** Vanilla JavaScript, HTML5, CSS3
*   **Audio Engine:** Tone.js
*   **Build Tool:** Vite
*   **Deployment:** Vercel
