# ♻️ RecycleLoop (리사이클루프)

**RecycleLoop**는 재활용품(유리병, 캔, 종이박스 등)의 소리를 악기로 활용하여 음악을 만들 수 있는 **웹 기반 단일 채널 루프 시퀀서 및 피아노롤(Piano Roll)** 입니다.

### 🔗 [Live Demo 보러가기](https://recycle-loop-maker.vercel.app)

<br>

## ✨ 주요 기능 (Features)

*   **직관적인 피아노롤 인터페이스:** 좌측에 음계(Pitch)가 표시된 무한 그리드 제공, 마우스를 이용한 노트 배치/길이 조절, 자유로운 화면 확대/축소(Zoom) & 이동(Pan) 기능
*   **플로팅 UI를 통한 간편한 제어:** 
    *   **악기 패널:** 악기 검색 기능 및 '멜로디/비트' 카테고리별 악기 선택 지원
    *   **컨트롤 패널:** 트랜스포트(재생/정지), BPM 및 볼륨(dB) 슬라이더 조절, 그리드 켜기/끄기, 화면 초기화 기능
    *   **노트 분할 (Subdivision):** 온음표부터 32분음표까지 정밀한 노트 스냅 설정 지원
*   **프로젝트 Import/Export (JSON):** 작업 중인 시퀀서 데이터를 로컬에 저장(Export)하고 언제든 다시 불러올 수 있는(Import) 기능

<br>

## ⌨️ 단축키 및 조작법 (Shortcuts & Controls)

작업 속도를 높여주는 다양한 단축키와 고급 편집 기능을 지원합니다.

*   **`Space`** : 영역 선택 모드 (Selection Mode) 켜기/끄기
*   **`Ctrl` + `Z`** : 실행 취소 (Undo)
*   **`Ctrl` + `Y`** : 다시 실행 (Redo)

**[영역 선택 모드 활성화 시]**
*   **`Ctrl` + `C`** : 선택된 영역의 노트 복사 및 붙여넣기
*   **`Delete`** : 선택된 영역의 노트 일괄 삭제
*   **악기 일괄 변경** : 영역을 선택한 상태에서 좌측 컨트롤 패널의 다른 악기를 클릭(토글)하면, 선택된 영역 내의 모든 노트가 해당 악기로 한 번에 변경됩니다.

<br>

## 🥁 제공되는 악기 목록 (Instruments)

UI 패널에서 제공되는 악기들은 크게 **멜로디(음계 배열)**와 **비트(타악기)** 두 가지 카테고리로 나뉘어 제공됩니다.

### 🎵 멜로디 (Melody)
*   **페트병 불기 (Blow Bottle):** 병 입구를 불어 내는 소리
*   **유리병 (Cling Glass):** 맑고 경쾌한 유리 타격음
*   **스틸 팬 (Steel Pan):** 깡통을 재활용한 멜로디 타악기 소리

### 🥁 비트 (Beat)
*   **페트병 구기기 (Squeeze Bottle):** 페트병을 구길 때 나는 질감 있는 소리
*   **종이박스 (Kick Box):** 묵직한 베이스 드럼(Kick) 역할의 상자 타격음
*   **캔 두드리기 (Hit Tin Can):** 금속 캔 타격음
*   **캔 따기 (Opening Can):** 탄산 캔을 개봉할 때 나는 경쾌한 효과음
